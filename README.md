# Movie-Api
## overview
This is a simple movie api with nodejs for applying CRUD operations.

## Features
  * Create, Read, Update, and Delete (CRUD) operations for movies items.

## Technology
  * NodeJs
  * ExpressJs
  * Mongodb&Mongoose
  
## How To Run 
 * run : npm install
 * add your config.env file
 * run : npm start
