# movie-Api
## overview
This is a simple movie api with nodejs for applying CRUD operations.

## Features
  * Create, Read, Update, and Delete (CRUD) operations for to-do items.

## Technology
  * NodeJs
  * ExpressJs
  * Mongodb&Mongoose
  
## How to Run 
 * run : npm install
 * add your config.env file
 * run : npm start
